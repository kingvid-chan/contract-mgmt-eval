# Iteration 0.0.3 — 合同模板功能

## 范围

新增合同模板功能模块。管理员可创建和管理合同模板（含占位符），用户创建合同时可从模板下拉选择快速填充。

- **owner_event**: e000031 (2026-06-18T12:20:41Z)
- **request_id**: eval-20260618-004-contract-template
- **kind**: feature（新功能迭代）

## 需求（来自老板原话）

1. 管理员可以创建和管理合同模板（模板名称、分类、合同内容模板含占位符如 `{{甲方}}`、`{{金额}}`、`{{日期}}`）
2. 用户创建合同时可从模板下拉选择快速填充——选择模板后自动填充合同标题、内容和结构字段
3. 模板管理页面（管理员专属）：模板列表、创建模板表单、编辑模板、删除模板（软删除）
4. 模板包含：模板名称、分类标签、预设的甲方/乙方默认值、合同正文模板（支持占位符）、预设金额范围
5. 创建合同页面增加「从模板创建」选项

## 业务模型变更

新增 ContractTemplate 实体：

- **name**: 模板名称（必填）
- **category**: 分类标签（如"采购合同"、"服务合同"）
- **party_a_default**: 甲方默认值
- **party_b_default**: 乙方默认值
- **content**: 合同正文模板，支持 `{{变量}}` 占位符
- **amount_min / amount_max**: 预设金额范围
- **is_deleted**: 软删除标记
- **created_by**: 创建者（外键 → users）

## 产品流程

1. 管理员创建模板：填写模板名称、分类、甲方/乙方默认值、正文模板（含占位符）、金额范围
2. 管理员管理模板：列表查看、搜索筛选、编辑更新、软删除
3. 用户创建合同：在新建合同页面选择「从模板创建」下拉 → 选择模板 → 表单自动填充 → 用户可修改 → 提交创建
4. 模板删除后不影响已通过模板创建的合同
